### About

I build low-level Windows tooling, Unity / IL2CPP reverse-engineering utilities, anti-cheat prototypes, and small desktop products under the **Encryptic** brand.

Focus areas: native C++, game runtime introspection, injectors & bases, and practical apps that ship.
